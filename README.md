# BERT Resume Analyzer

A machine learning project that uses BERT to analyze resumes, extract important skills, classify candidate-job fit, and provide resume improvement insights.

## Features

- Resume text extraction
- Skill and keyword detection
- BERT-based resume classification
- Job description matching
- Resume scoring system
- Improvement suggestions

## Tech Stack

- Python
- PyTorch
- Hugging Face Transformers
- BERT
- scikit-learn
- FastAPI
- Streamlit
